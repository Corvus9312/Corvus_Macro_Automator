# Corvus Macro Automator

使用 `uv` + `PyQt6` 建立的按鍵精靈專案，提供多巨集管理、圖形化編輯、全域熱鍵錄製與主畫面快速播放。

## 目前功能

- 多巨集清單管理（新增、編輯、重新命名、刪除）
- 巨集資料持久化：`macros.json`
- 匯入/匯出單一巨集（JSON）
- 主畫面直接播放/停止（可用按鈕或熱鍵）
- 主畫面清單顯示每個巨集的：
  - 開始延遲
  - 播放速度
  - 播放方式
  - 重複間隔
  - 事件數量
- 編輯頁可錄製全域鍵盤/滑鼠事件
- 編輯頁可手動新增指令：
  - 聚焦視窗
  - 等待（時/分/秒/毫秒）
  - 移動滑鼠（含即時座標）
  - 滑鼠點擊（左/右鍵）
  - 鍵盤輸入
  - 鍵盤組合鍵（含按壓時間 ms）
- 指令清單支援：
  - 雙擊編輯
  - 多選刪除（`Delete`）
  - 多選複製（`Ctrl+C`）
  - 上移/下移調整順序
- 播放模式支援：
  - 不重複
  - 重複次數
  - 重複至時間（`YYYY-MM-DD HH:MM:SS`）
  - 到手動停止

## 熱鍵規則（F5 / F6）

- 主畫面（巨集清單頁）
  - `F5`：開始播放目前選取巨集
  - `F6`：停止播放
- 編輯頁
  - `F5`：開始錄製
  - `F6`：停止錄製

## 安裝與執行

1. 安裝依賴

   ```bash
   uv sync
   ```

2. 啟動程式

   ```bash
   uv run python main.py
   ```

   或：

   ```bash
   uv run corvus-macro
   ```

## 技術堆疊

- Python `>=3.11`
- `PyQt6`
- `PyAutoGUI`
- `pynput`

## 注意事項

- 本工具會控制全域鍵盤/滑鼠輸入，請先確認目標視窗與操作環境。
- 錄製與播放前，建議先關閉不必要視窗，避免誤操作。
- 遊戲或特權視窗可能有額外防護機制，若輸入未生效可嘗試以系統管理員執行。
