# Corvus Macro Automator

使用 `uv` + `PyQt6` 建立的按鍵精靈專案，提供多巨集管理、圖形化編輯、全域熱鍵錄製與主畫面快速播放。

## 目前功能

- 多巨集清單管理（新增、編輯、重新命名、刪除）
- 巨集資料持久化：`macros.json`
- 匯入/匯出單一巨集（JSON）
- 主畫面直接播放/停止（可用按鈕或熱鍵）
- 主畫面排程設定（最多 3 個腳本）
- 主畫面清單顯示每個巨集的：
  - 開始延遲
  - 播放速度
  - 播放方式
  - 重複間隔
  - 事件數量
- 編輯頁可錄製全域鍵盤/滑鼠事件
- 編輯頁可手動新增指令：
  - 聚焦視窗
  - 等待（時/分/秒/毫秒）
  - 移動滑鼠（含即時座標）
  - 滑鼠點擊（左/右鍵）
  - 鍵盤輸入
  - 鍵盤組合鍵（含按壓時間 ms）
- 指令清單支援：
  - 雙擊編輯
  - 多選刪除（`Delete`）
  - 多選複製（`Ctrl+C`）
  - 上移/下移調整順序
- 播放模式支援：
  - 不重複
  - 重複次數
  - 重複至時間（`YYYY-MM-DD HH:MM:SS`）
  - 到手動停止

## 排程功能（主畫面）

- 最多可設定 3 個排程腳本（腳本1、腳本2、腳本3）。
- 每個腳本使用下拉選單選擇巨集，選單包含 `---請選擇---`：
  - 若維持 `---請選擇---`，該腳本會被忽略（不初始化、不啟動計時器）。
- 每個腳本可設定 `每隔(分鐘)`，預設為 `1` 分鐘。
- 點擊 `開始排程` 後流程如下：
  1. 先依序執行一次：腳本1 -> 腳本2 -> 腳本3（僅執行有選擇的腳本）。
  2. 初始化播放完成後，才啟動各自的循環計時器。
- 若某腳本定時觸發時，另一腳本正在播放：
  - 會自動加入佇列，等待前一個腳本播放結束後接續執行。
- `停止播放`（或 `F6`）會停止目前播放並清空待執行佇列。

## 熱鍵規則（F5 / F6）

- 主畫面（巨集清單頁）
  - `F5`：開始播放目前選取巨集
  - `F6`：停止播放
- 編輯頁
  - `F5`：開始錄製
  - `F6`：停止錄製

## 安裝與執行

1. 安裝依賴

   ```bash
   uv sync
   ```

2. 啟動程式

   ```bash
   uv run python main.py
   ```

   或：

   ```bash
   uv run corvus-macro
   ```

## 技術堆疊

- Python `>=3.11`
- `PyQt6`
- `PyAutoGUI`
- `pynput`

## 注意事項

- 本工具會控制全域鍵盤/滑鼠輸入，請先確認目標視窗與操作環境。
- 錄製與播放前，建議先關閉不必要視窗，避免誤操作。
- 遊戲或特權視窗可能有額外防護機制，若輸入未生效可嘗試以系統管理員執行。
